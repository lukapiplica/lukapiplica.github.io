### Hvala na posjeti!

<img src="/assets/images/popnord/screen.jpg" alt="img" align="center" width="400px">

Ovo su moji trenutni dotfile-ovi Pop! OS Linux distribucije.

Instalacija će vam pomoći da napravite isti izgled kao na slikama, tu imate uputstvo korak po korak.

Sistemske Specifikacije:

+ **OS**: [Pop!_OS](https://pop.system76.com/) 
+ **WM**: [BSPWM](https://github.com/baskerville/bspwm) 
+ **Keyboard Deamon**: [Sxhkd](https://github.com/baskerville/sxhkd)
+ **Bar**: [Polybar](https://github.com/polybar/polybar)
+ **Compositor**: [Picom](https://github.com/ibhagwan/picom)
+ **App Launcher**: [Rofi](https://github.com/davatorium/rofi)
+ **Terminal**: [Alacritty](https://github.com/alacritty/alacritty)
+ **Shell**: [Oh-my-ZSH](https://ohmyz.sh/)
+ **Shell theme**: [Powerlevel10k](https://github.com/romkatv/powerlevel10k)
+ **Text Editor**: [Vim-Airline](https://github.com/vim-airline/vim-airline)
+ **Lockscreen**: Slim and Slimlock
+ **Notifications**: [Dunst](https://dunst-project.org/)
+ **GTK Theme**: [Nordic_Dark](https://www.gnome-look.org/p/1267246/)
+ **Icon Theme**: [Flatery_Dark](https://www.gnome-look.org/s/Gnome/p/1332404)
+ **Cursor Theme**: [Oreo_Blue](https://www.gnome-look.org/s/Gnome/p/1360254/)
+ **Font**: [Source_Code_Pro](https://fonts.google.com/specimen/Source+Code+Pro)
+ **Browser**: [Firefox](https://www.mozilla.org/en-US/firefox/new/)
+ **Browser Theme**: [Minimal_Functional_Fox](https://github.com/mut-ex/minimal-functional-fox)

## Instalacija

Da bismo počeli sa pravljenjem ovog izgleda, pretpostavljam da imate friško instaliran Pop! OS.

*Ako vas zanima šta koja aplikacija radi Wikipedia 'e vam pomoći.*

1. Updejtanje repozitorija i upgrejdanje sistema: 

      ```shell
      sudo apt update
      sudo apt upgrade
      ```

2. Instalacija **BSPWM Tilling Window Manager-a**:

   Prvo moramo instalirati nekoliko potrebnih tzv. dependencies-a koji su nam potrebni da bi nastavili dalje sa instalacijom.

   + Instalacija dependancy-a:

     ```shell 
     sudo apt install build-essential git vim xcb libxcb-util0-dev libxcb-ewmh-dev libxcb-randr0-dev libxcb-icccm4-dev libxcb-keysyms1-dev libxcb-xinerama0-dev libasound2-dev libxcb-xtest0-dev libxcb-shape0-dev
     ```
     

   + Kloniranje repozitorije:

     ```shell
     cd ~/Downloads
     git clone https://github.com/baskerville/bspwm.git
     ```
   + Kompajliranje i installacija BSPWM-a:

     ```shell
     cd bspwm
     make
     sudo make install
     ``` 
   + Kopiranje BSPWM konfiguracijskih fajlova:

     ```shell
     mkdir ~/.config/bspwm
     cp examples/bspwmrc ~/.config/bspwm
     chmod +x ~/.config/bspwm/bspwmrc
     cd ..
     ``` 


3. Instalacija **keyboard deamon-a**:

   Da bismo mogli koristiti kratice na tastaturi da nam naprimjer otvore terminal, neku drugu aplikaciju itd. moramo koristiti tzv. keyboard deamon. To je proces koji radi u pozadini i daje instrukcije BSPWM Tilling Window Manager-u šta da uradi ako netko pritisne određenu kraticu na tastaturi.

      *Ako vas zanima šta koja kratica radi po mojim postavkama Kratice će vam pomoći.*

      + Kloniranje repozitorije:

         ```shell
         cd ~/Downloads
         git clone https://github.com/baskerville/sxhkd.git
         ```

      + Kompajliranje i instalacija sxhkd:
      
         ```shell
         cd sxhkd
         make 
         sudo make install
         ```

      + Kopiranje sxhkd konfiguracijskih fajlova:

         ```shell
         mkdir ~/.config/sxhkd
         cp ../bspwm/examples/sxhkdrc ~/.config/sxhkd
         cd ..
         ```

      - **NAPOMENA:** Ako budete mijenjali išta u sxhkdrc fajlu, obratite pažnju o tome koji vam je terminal postavljen, ako koristite obični terminal od Pop! OS-a onda morate promijeniti u `gnome-terminal` pod terminal emulator.


4. Instalacija **Polybar**-a:

   + Instalacija potrebnih dependancy-a: 

      ```shell
      sudo apt install cmake cmake-data pkg-config python3-sphinx libcairo2-dev libxcb1-dev libxcb-util0-dev libxcb-randr0-dev libxcb-composite0-dev python3-xcbgen xcb-proto libxcb-image0-dev libxcb-ewmh-dev libxcb-icccm4-dev libxcb-xkb-dev libxcb-xrm-dev libxcb-cursor-dev libasound2-dev libpulse-dev libjsoncpp-dev libmpdclient-dev libcurl4-openssl-dev libnl-genl-3-dev
      ```

   + Kloniranje repozitorija: 

      ```shell
      cd ~/Downloads
      git clone --recursive https://github.com/polybar/polybar
      ```

   + Kompajliranje i instalacija polybar-a:

      ```shell
      cd polybar
      mkdir build
      cd build
      cmake ..
      make -j$(nproc)
      sudo make install
      ```

5. Instalacija **Picom**-a:

   + Instalacija potrebnih dependancy-a:
   
      ```shell
      sudo apt install meson libxext-dev libxcb1-dev libxcb-damage0-dev libxcb-xfixes0-dev libxcb-shape0-dev libxcb-render-util0-dev libxcb-render0-dev libxcb-randr0-dev libxcb-composite0-dev libxcb-image0-dev libxcb-present-dev libxcb-xinerama0-dev libpixman-1-dev libdbus-1-dev libconfig-dev libgl1-mesa-dev  libpcre2-dev  libevdev-dev uthash-dev libev-dev libx11-xcb-dev
      ```

   + Kloniranje repozitorije: 

      ```shell
      cd ~/Downloads
      git clone https://github.com/ibhagwan/picom.git
      ```

   + Pravljenje picom-a sa Ninja-om:

      ```shell
      cd picom
      git submodule update --init --recursive
      meson --buildtype=release . build
      ninja -C build
      ```

   + Instalacija picom kompozitora: 

      ```shell 
      sudo ninja -C build install
      cd ..
      ```

6. Instalacija **Rofi**-a: 

   ```shell
   sudo apt install rofi
   ```

7. Instalacija **Alacritty** terminala: 

   ```shell
   sudo apt install alacritty
   ```

8. Kloniranje ove repozitorije

   ```shell
   cd ~/Downloads

   ```

9. Postavljanje alacritty teme: 

   ```shell
   mkdir ~/.config/alacritty
   cp dots/alacritty/alacritty.yml ~/.config/alacritty/
   ```

   + **NAPOMENA**: Ako dobijete error: `GLSL 3.30 is not supported`, uradite ovo:

      ```shell
      nano /usr/share/applications/com.alacritty.Alacritty.desktop
      ```

      Promjenite `Exec=alacritty` u `Exec=bash -c "LIBGL_ALWAYS_SOFTWARE=1 alacritty"`.

+ **NAPOMENA:** Nakon ovog koraka spremni ste da se prijavite u BSPWM, prvo se trebate odjaviti sa trenutnog desktop interfejsa, a zatim u donjem desnom ćošku kliknete na ⚙️ ikonicu i izaberete BSPWM.

## Wikipedia

Nakon što smo napravili ovu konfiguraciju BSPWM-a vrijeme je da pojasnimo šta koja aplikacija radi. Unutar `~/.config/` direktoriji možete pronaći sve fajlove koji su vam potrebni za rad BSPWM-a.

+ 🧠 `[BSPWM]`

   BSPWM je tilling window manager koji za razliku od ostalih desktop interfejsa, svaki novi otvoren prozor reda po nekom algoritmu.

   Ako imamo naprimjer dva prozora oni će biti otvoreni preko čitavog ekrana, i ne mogu se nalaziti jedan preko drugog kao npr. u Windows ili GNOME interfejsima. Teoretski to nam efektivno koristi prostor na ekranu tako da čitav ekran bude popunjen. Ako nam ponestaje prostora na prvoj radnoj površini, možemo preći na drugu radnu površinu pomoću kratice na tastaturi ili mišem i tako nastaviti koristiti ovaj operativni sistem. 

+ ⌨️ `[SXHKD]`

   SXHKD nam je keyboard deamon. 

   On nam radi kao proces u pozadini i govori nam BSPWM-u koju aplikaciju otvoriti ako korisnik pritisne određenu kraticu na tastaturi.

+ 🍫 `[Polybar]`

   Polybar nam je statusna traka na vrhu našeg desktopa.

   Ona nam služi da prikaže notifikacije, datum i vrijeme, internet, bateriju, meni za gašenje i restartovanje operativnog sistema i najbitnije od svega radne prostore (workspaces) od BSPWM-a.

+ 📚 `[Picom]`

   Picom je kompozitor, on nam služi kao proces koji postavlja zaobljene rubove na svakoj aplikaciji koju otvorimo.

+ 🤓 `[Rofi]` 

   Rofi služi kao proces koji nam otvara druge aplikacije, najviše koristan za pokretanje GTK aplikacija (aplikacija sa korisničkim interfejsom).

+ 💻 `[Alacritty]` terminal.

   Alacritty nam služi kao emulator terminala tj. zamjena standardnome GNOME terminalu.

   Njega koristimo najviše zbog toga što nam pruža mogućnost kodiranja boljeg izgleda.

+ 🐚 `[Oh-My-ZSH]` shell.

   Oh-My-ZSH je framework koji nam služi za lakšim upravljanjem ZSH konfiguracija.

   U ovome slučaju najviše se koristi za dodavanje tema u terminal koji ima `ZSH shell`.

   `ZSH (Zshell)` je izabran zato što je noviji i većina updejta od ostalih Linux distribucija već prelazi na njega u odnosu na `Bash shell`.

    + Za postavljanje tema smo koristili `Powerlevel10k` konfiguraciju.


+ 📋 `[Vim]` text editor

  Kao što mnogi ljudi koriste notepad na Windows operativnim sistemima, mi koristimo nano i vim.

  Nano text editor je puno lakši za korištenje, dok Vim je napravljen više za programere zbog njegovih kratica. 

   + Vim Airline je tema koja nam daje drugačiji izgled od običnog Vim-a.


+ 📺 `[Slim]` lockscreen

   Slim nam je zaslon zaključavanja (lockscreen). 

   Slimlock nam je tema koju smo koristili za zaslon zaključavanja.

+ 💬 `[Dunst]` notifikacije.

   Dunst nam je proces koji radi u pozadini i glavna svrha mu je da prikazuje notifikacije na statusnoj traci (Polybar-u).

   Prednost dunst-a je ta što pruža mogućnost kodiranja tema za izgled i mnogih drugih funkcija kao npr. koje notifikacije da prikaže itd.

+ 🕶️ `[Nordic Dark]` GTK tema.

   Nordic Dark je GTK tema (tema za aplikacije sa korisničkim interfejsom) koja nam daje tamni izgled aplikacijama koje imaju korisnički interfejs.

+ 🔷 `[Flattery Dark]` tema za ikonice.

   Samo ime kaže, aplikacije koje imaju korisnički interfejs većinom imaju ikonice na koje možemo kliknuti mišem, ova tema nam pruža drugačiji izgled ikonicama.

+ 🔵 `[Oreo Blue]` kursor tema.

   Ova tema nam daje plavi izgled kursoru, ubačena je samo radi izgleda kursora i zato što se slaže sa svim ostalim bojama ovog operativnog sistema.

+ 🔤 `[Source Code Pro]` font.

   Samo ime kaže font ovog operativnog sistema. 

+ 🔥🦊 `[Firefox]` web preglednik.

   Firefox koristim jer je jedan od po mome mišljenu najboljih web preglednika koji je orijentisan ka privatnosti.

+ 🦊 `[Minimal Functional Fox]` firefox tema.

   Ova tema nam je ovdje najviše radi izgleda, ali i zbog toga što je minimalna. 

+ ⏰ `tty-clock` 

   Tty-clock nam je aplikcija koja ima svrhu da u terminalu pokaže vrijeme i datum. 

+ 🎚️ `CAVA` audio visualizer.

   Cava nam je aplikacija koja služi kao audio vizualizator.

+ 📈 `HTop` 

   HTop aplikacija nam služi da prikaže koji procesori trenutno rade i koliko RAM memorije oni uzimaju kao i mnogo drugih stvari. Zamislite HTop kao Task Manager u Windows operativnom sistemu.

+ 😎 `CMatrix` 

   CMatrix nam je aplikacija koja je ovdje samo radi izgleda i njezina svrha je da u otvorenom terminalu ispisuje neki text kojem je cilj da daje prikaz hakovanja iz filma Matrix-a. 

+ 🖼️ `SXIV` 

   SXIV je image viewer aplikacija to jeste aplikacija koja nam otvara slike.

+ 🤠 `Ranger` 

   Ranger je aplikacija koja nam omogućava da u terminalu vidimo i pristupimo svim fajlovima na računaru. 

+ 🗄️ `Pfetch`

   Pfetch aplikacija ima svrhu da prikaže koji operativni sistem koristimo, koji nam je laptop/računar, koja je trenutna verzija kernela, koliko nam uređaj trenutno radi (uptime), koliko ukupno paketa imamo i koliko imamo RAM memorije.

+ ⚙️ `Chafa`

   Chafa je aplikacija koja nam ispisuje sliku u terminalu, time postižemo retro/ASCII izgled neke slike. 

+ 🐍 `Snake`

   Snake aplikacija je u ovome slučaju stara snake igra sa Nokia 3310 telefona koja je malo izmjenjena u smislu boja i prilagođena je da se igra u terminalu. 

+ 🍭 `Lollypop` 

   Lollypop je aplikacija za muziku, preko nje puštamo pjesme. 


## Kratice 

Kao <kbd>super</kbd> koristimo Windows tipku na tastaturi. Super nam je najvažnija tipka na tastaturi jer preko nje dajemo instrukcije SXHKD keyboard deamon-u.

#### Tastatura

| Tipke | Akcija |
| --- | --- |
| <kbd>super + enter</kbd> | Otvara terminal |
| <kbd>super + space</kbd> | Otvara Rofi preko kojeg otvaramo GTK aplikacije |
| <kbd>super + escape</kbd> | Ponovno učitaje SXHKD i njegove konfiguracijske fajlove |
| <kbd>super + alt + r</kbd> | Restartuje BSPWM Tilling Window Manager |
| <kbd>super + w</kbd> | Gasi trenutnu aplikaciju |
| <kbd>super + [1-0]</kbd> | Mijenja trenutnu radnu površinu |
| <kbd>super + g</kbd> | Mijenja trenutni prozor koji je manji na površinu većeg prozora, dok onaj veći prozor stavlja na poziciju manjeg |
| <kbd>super + m</kbd> | Otvara aplikaciju preko čitave radne površine |
| <kbd>super + [h,j,k,l]</kbd> | Pomjera focus na neki drugi prozor |
| <kbd>super + alt + [h,j,k,l]</kbd> | Pomjera prozore prema van |
| <kbd>super + alt + shift + [h,j,k,l]</kbd> | Pomjera prozore prema unutra |
| <kbd>super + s</kbd> | Stavlja prozore u tzv. float mode tj. mogućnost gdje možemo stavljati prozor na prozor kao u Windows operativnim sistemima |
| <kbd>super + ctrl + [strelice]</kbd> | Pomjera prozor koji je u float-u |



### Hvala puno na izdvojenom vremenu 💙
